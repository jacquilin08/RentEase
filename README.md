# RentEase
**RentEase – Premium Furniture & Appliance Rental Platform**

RentEase is a modern web-based rental platform designed to simplify the process of renting furniture and home appliances. The application provides users with an affordable and flexible alternative to purchasing expensive household products by offering short-term and long-term rental plans.

The platform features a user-friendly interface that allows customers to browse a wide range of furniture and appliances, view product details, select rental tenures, add items to a shopping cart, and place rental orders seamlessly. Users can also manage their active rentals, track rental history, submit maintenance requests, and monitor order status through a personalized dashboard.

An integrated administrative dashboard enables efficient management of products, inventory, customers, rentals, damage claims, service areas, and business analytics. The system provides real-time insights into inventory utilization, rental performance, revenue statistics, and customer engagement, helping administrators make informed business decisions.

The project is developed using HTML, CSS, and JavaScript with a responsive and modern user interface that delivers a smooth user experience across desktop and mobile devices. RentEase demonstrates key concepts of e-commerce, rental management, inventory tracking, and dashboard analytics while emphasizing usability, scalability, and visual appeal.

**Key Features:**

* User Registration and Login System
* Product Catalog with Search and Filtering
* Shopping Cart and Rental Checkout
* Flexible Rental Tenure Selection
* Active Rental Tracking
* Rental History Management
* Maintenance Request System
* Admin Dashboard and Analytics
* Inventory Management
* Damage Claim Monitoring
* Service Area Management
* Responsive and Modern UI Design

**Technologies Used:**

* HTML5
* CSS3
* JavaScript (ES6)
* Responsive Web Design
* Local Data Management

RentEase aims to make quality furniture and appliances accessible through a convenient rental ecosystem while providing an efficient management solution for rental service providers.
